# Transações

## O que é essa área?

A área de transações é onde você registra todas as movimentações do seu dinheiro.

Cada transação representa uma entrada (receita) ou uma saída (despesa).

---

## Pra que serve?

Essa área é a base de todo o sistema.

É através das transações que o sistema consegue:

* Calcular seu saldo
* Gerar gráficos
* Criar análises
* Mostrar alertas
* Acompanhar sua evolução financeira

Sem transações, o sistema não tem dados para funcionar.

---

## Como usar?

### Adicionar uma transação

1. Clique em **"Adicionar nova transação"**
2. Preencha os campos:

   * Descrição
   * Valor
   * Tipo (receita ou despesa)
   * Categoria
   * Data
3. Clique em **salvar**

---

### Editar uma transação

1. Vá até a lista de transações
2. Clique em **"Editar"**
3. Atualize as informações
4. Salve as alterações

---

### Excluir uma transação

1. Clique em **"Excluir"** ao lado da transação
2. Confirme a ação

---

## O que cada campo significa?

* **Descrição** → nome da transação (ex: mercado, salário)
* **Valor** → quanto foi ganho ou gasto
* **Tipo**:

  * Receita → dinheiro que entrou
  * Despesa → dinheiro que saiu
* **Categoria** → classificação do gasto (ex: alimentação, transporte)
* **Data** → quando a transação aconteceu

---

## Filtros e busca

Você pode filtrar suas transações por:

* Mês
* Ano
* Palavra-chave (busca por descrição ou categoria)
* Ordenação (data, valor ou categoria)

Isso ajuda a encontrar informações específicas rapidamente.

---

## Exportação de dados

O sistema permite exportar suas transações em dois formatos:

* **CSV** → ideal para abrir no Excel ou Google Sheets
* **Excel (.xlsx)** → formato pronto para análise

---

## Como funciona por trás?

Cada transação é salva no banco de dados com:

* ID do usuário
* Tipo (receita ou despesa)
* Valor
* Categoria
* Data

O sistema então utiliza esses dados para:

* Calcular totais
* Gerar gráficos
* Criar análises

---

## Por que isso é importante?

Se você não registra seus gastos, você perde o controle do seu dinheiro.

Essa área garante que tudo fique organizado, permitindo que você entenda exatamente para onde seu dinheiro está indo.
